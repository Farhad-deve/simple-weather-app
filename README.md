# Simple Weather App

## Project Description

This is simple weather app built with Vanilla JS.

![The screenshot of web:](/assets/screenshot.png)

[Visit Website 🚀](https://farhad-deve.github.io/simple-weather-app/)

## Technologies used:
- HTML5 (Semantic HTML structure)
- CSS3 (Several basic styles)
- JavaScript (Weather logic & data from [Weather API](https://openweathermap.org/api/one-call-3?collection=one_call_api_3.0))
- Tailwind CSS

## Users are able to:
- See the current temperature, weather details (humidity, wind speed e.tc) depending on current location and entered city or country
